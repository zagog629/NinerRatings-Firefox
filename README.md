# NinerRatings for Firefox 🦊⛏️
As Seen on Niner Times!

> **Credit & Attribution:** This is a Firefox port of the original NinerRatings Chrome extension. **All credit for the original creation, logic, and design goes entirely to the original developer.** You can find the original Chrome version [right here](https://github.com/ausmango/NinerRatings).

Tired of manually looking up every professor to see their ratings? I built NinerRatings to help Charlotte students and their registration process. Instead of frantically switching between tabs to pick the best option, this extension directly embeds the data right in your portal!

## Features
- 🔗 Embedded hyperlink in each professor name for fast look-up

- 🟢🟡🔴 Visibility dots to show rating color data on initial viewing

- Hover badge showing important stats directly from RMP's website

- Supports multiple professors per course selection

- 3-day Chrome Storage caching for near-instant repeat lookups

## Disclaimer
Rate My Professors is an unofficial website that is utilized by students. All reviews are to be viewed with skepticism.

- **AVAILABILITY** - This extension only works on Charlotte's self service registration portal/browse class search, and no other university.

- **NAME MISMATCHES** - Some professor names differ from Self Service compared to their RMP profiles. This must be fixed manually so some professors may show as 'N/A: no data found,' even if they have data.

- **LOW DATA** - Professors with under 10 ratings may not be representative of actual quality.

- **RMP SITE** - IF RMP changes their endpoint, the extension may temporarily be down until fixed.
